Verilog HDL Vivado RISC-V

RISC-V Pipelined ALU Design

Designed a 4-stage pipelined Arithmetic Logic Unit (ALU) in Verilog HDL, matching the RV32I execution specifications. Registers separate the Fetch, Decode, Execute, and Writeback stages, minimizing combinational path delays and preventing hazards.

  • Implemented 8 core instructions: ADD, SUB, AND, OR, XOR, SHL, SHR, SLT.
  • Integrated data-forwarding paths and stall control to resolve Read-After-Write (RAW) data hazards.
  • Validated setup/hold margins at stage boundaries using Vivado post-implementation timing analysis.

SystemVerilog Vivado Logic Synthesis

Parameterized N-Bit Barrel Shifter

Designed a parameterized N-bit barrel shifter supporting Logical Left (LSL), Logical Right (LSR), and Arithmetic Right (ASR) shifts. Utilizing a cascaded MUX-tree structure, the design shifts any bit depth in a single clock cycle, replacing slow iterative shifting loops.

  • Implemented a log2(N) stage 2-to-1 multiplexer tree for single-cycle operations.
  • Verified operation across 30+ corner-cases (zero-shift, sign-extension boundaries, max shifts).
  • Minimized propagation delay, compiling successfully without any inferred latches.

SystemVerilog Hardware Accelerator Systolic Array

Systolic Matrix Multiplier Unit

Developed an FPGA-optimized 4x4 Systolic Array Matrix Multiplier in SystemVerilog. Optimized for edge AI and tensor operations, it achieves high data reuse by propagating inputs through localized processing elements (PEs).

  • Implemented 16 PEs containing 16-bit MAC (Multiply-Accumulate) units and local registers.
  • Reduced RAM bandwidth demands by loading data inputs and propagating values horizontally/vertically.
  • Synthesized on a Xilinx Artix-7 board, meeting target clock frequencies at minimal power.
