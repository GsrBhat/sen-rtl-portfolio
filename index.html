Verilog HDL Xilinx Vivado RISC-V ISA

RISC-V Pipelined ALU Design

Designed and verified a 4-stage (Fetch → Decode → Execute → Writeback) pipelined ALU in Verilog HDL, matching the RV32I execution specifications. Used pipeline registers at stage boundaries to eliminate combinational feedback, reduce critical path delay, and improve clock frequency.

  • Implemented 8 operations: ADD, SUB, AND, OR, XOR, SHL, SHR, and SLT.
  • Integrated data-forwarding paths and stall control logic to resolve Read-After-Write (RAW) data hazards.
  • Validated timing margins (setup/hold) and path delays using Xilinx Vivado directed testbenches.

Verilog HDL Xilinx Vivado Mux Tree Logic

Parameterized N-Bit Barrel Shifter

Designed a high-speed parameterized N-bit barrel shifter supporting Logical Left (LSL), Logical Right (LSR), and Arithmetic Right (ASR) operations. Built using a cascaded 2-to-1 MUX tree ($log_2 N$ levels) to complete shifts in a single clock cycle, eliminating serial iteration loops.

  • Supports parameterized data widths (N = 8, 16, 32, 64) with opcode control.
  • Verified operation across corner-cases including maximum-shifts, zero-shifts, and sign extensions.
  • Optimized gate depth, ensuring efficient FPGA LUT mapping with zero inferred latches.

SystemVerilog FPGA Prototyping Systolic Array

Systolic Array Matrix Multiplier

Designed and prototyped an FPGA-optimized 2D systolic array core in SystemVerilog for high-speed matrix multiplication. Designed processing element (PE) nodes with Multiply-Accumulate (MAC) circuits, enabling local registers to maximize data reuse.

  • Implemented 16 processing element nodes (4x4 grid) with 16-bit multipliers and accumulators.
  • Arranged a horizontal/vertical wavefront data feed to reduce external memory access bottlenecks.
  • Mapped design constraints to a Xilinx Artix-7 board, meeting target clock timing and floorplan grids.
